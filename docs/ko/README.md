# Gemini CLI 워크숍

> **기업 개발자를 위한 실습 교육** — Gemini CLI의 에이전트 도구, 플랜 모드, 확장 가능한 에이전트 시스템을 활용하여 에이전트 기반 코딩, 레거시 현대화, DevOps 자동화를 마스터하세요.
>
> *최종 업데이트: 2026-05-05 · [gemini-cli 저장소 기준 검증됨](https://github.com/google-gemini/gemini-cli)*

---

## 워크숍 진행 순서

이 워크숍은 **3가지 점진적 사용 사례**로 구성됩니다. 각 사례는 독립적이지만 이전 단계에서 습득한 기술을 기반으로 합니다:

![워크숍 흐름](../assets/workshop-flow.png)

---

## 사용 사례 요약

### [1. SDLC 생산성 향상](sdlc-productivity.md)
첫 설치부터 컨텍스트 엔지니어링, Conductor를 활용한 스펙 기반 개발, 거버넌스 가드레일까지 엔터프라이즈급 개발자 워크플로우를 구축합니다.

### [2. 레거시 코드 현대화](legacy-modernization.md)
플랜 모드, 커스텀 서브에이전트, 스킬, 체크포인팅을 사용하여 레거시 .NET Framework 4.8 앱을 .NET 8 on Cloud Run으로 마이그레이션합니다.

### [3. 에이전트 기반 DevOps 오케스트레이션](devops-orchestration.md)
파이프라인 실패를 진단하고, 수정 사항을 생성하며, PR을 제출하고, 팀에 알림을 보내는 CI/CD 자동화를 구축합니다.
