## 工作坊流程

本工作坊由 **3 个递进式用例** 组成。每个用例独立完整，但建立在前一阶段习得的技能之上：

## 用例概览

### [1. SDLC 生产力提升](sdlc-productivity.md)
从首次安装到上下文工程、使用 Conductor 进行规范驱动开发，以及治理护栏，构建企业级开发者工作流。

### [2. 遗留代码现代化](legacy-modernization.md)
使用 Plan Mode、自定义子代理、技能和检查点，将遗留 .NET Framework 4.8 应用迁移到 Cloud Run 上的 .NET 8。

### [3. 智能体 DevOps 编排](devops-orchestration.md)
构建 CI/CD 自动化，诊断流水线故障、创建修复、提交 PR 并通知团队。
