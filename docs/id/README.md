## Ringkasan Kasus Penggunaan

### [1. Peningkatan Produktivitas SDLC](sdlc-productivity.md)
Bangun alur kerja developer tingkat enterprise dari instalasi pertama hingga context engineering, pengembangan berbasis spesifikasi dengan Conductor, dan guardrail tata kelola.

### [2. Modernisasi Kode Legacy](legacy-modernization.md)
Migrasikan aplikasi legacy .NET Framework 4.8 ke .NET 8 di Cloud Run menggunakan Plan Mode, subagen kustom, skills, dan checkpointing.

### [3. Orkestrasi DevOps Berbasis Agen](devops-orchestration.md)
Bangun otomatisasi CI/CD yang mendiagnosis kegagalan pipeline, membuat perbaikan, mengirim PR, dan memberi notifikasi tim.
